# Converters Module

Video encoding converters supporting hardware and software modes.

## Overview

The converters module provides H.264 to H.265 conversion using:

- **Hardware Converter**: Apple VideoToolbox (fast, efficient)
- **Software Converter**: libx265 (high quality, flexible)
